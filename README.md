# 📦 NovaInventory

A Java-based Inventory and Billing System built using Object-Oriented Programming concepts.

## Features

- Add Inventory Items
- Search Items
- Delete Items
- View All Items
- Generate Customer Bills
- CSV Data Storage
- Menu Driven CLI

## Technologies

- Java
- OOP
- Collections
- File Handling
- IntelliJ IDEA
- Git

## Project Structure

```
app
model
repository
service
util
```
